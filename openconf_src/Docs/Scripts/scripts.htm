Скрипты
OpenConf внедряет в среду Конфигуратора технологию "ActiveScripting", позволяющую исполнять скрипты на любом из установленных в системе ActiveScript-языков. Наиболее распространенные это VBScript и JScript, установленные практически на любом компьютере, где стоит Конфигуратор.
Скрипты представляют из себя файлы, размещаемые в каталоге config\scripts (см. "Установка")
