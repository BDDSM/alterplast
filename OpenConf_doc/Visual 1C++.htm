Плагин Visual 1C++ для платформы OpenConf

Плагин предоставляет возможность быстрой навигации по классам 1С++ и является оболочкой (проводником) для работы с классами 1С++ в среде Конфигуратора.

Авторы плагина: Александр Орефков, Артур Аюханов, Федор Езеев, Алексей Диркс.

Плагин можно скачать здесь (версия 1.0.0.13 beta, размер архива 49 526 байт).

Чтобы установить плагин, скопируйте файл VIS1CPP.DLL в подкаталог \BIN\Config\ установки 1С:Предприятия. Если 1С:Предприятие было запущено, перезапустите его. Если установка прошла успешно, в окне конфигурации вы должны увидеть дополнительную вкладку "1C++".

Удобнее всего использовать плагин в виде панели, аналогичной синтакс-помощнику. Для этого в меню "OpenConf" - "Настройка панелей..." добавьте новую панель, а затем переместите в неё доступный плагин "1С++". Для вступления изменений в силу перезапустите Конфигуратор. Панель будет доступна в любой момент через контекстное меню, которое появляется при щелчке правой кнопки мыши по любой панели инструментов Конфигуратора. В панели будет отображена иерархия используемых в конфигурации классов 1С++ с их свойствами и методами.

Возможности плагина:

  • Быстрый переход к определениям методов и свойств классов двойным щелчком. Такой же переход к определениям самих классов в целом и к файлу объявлений классов .prm.
  • Есть возможность сохранения модулей классов из контекстного меню в дереве.
  • Просмотр свойств элементов дерева через контекстное меню. Здесь можно просмотреть каталог базы данных, программы, путь к файлу объявлений классов .prm, путь к файлу каждого класса.
  • Наименование любого элемента дерева копируется в буфер, а также легко перетаскивается мышью в окно текстового редактора.
  • Из контекстного меню доступен синтакс-контроль каждого класса как с глобальным модулем, так и без.
  • Команда "Обновить" контекстного меню корневого узла дерева формирует заново всё дерево классов.
  • Флажок контекстного меню "Группировать по типу" группирует все вложенные элементы по типу (иначе элементы располагаются в алфавитном порядке).
